# jsx-autoedit

a Vim plugin to replace jsx tag with interactive UI.

# Demo


![](https://github.com/acro5piano/jsx-autoedit/blob/master/demo.gif)

# Install

```
Plug 'acro5piano/jsx-autoedit'
```

# Example setting

In your `.vimrc`,

```vim
nnoremap <Leader>rt :JSXReplaceTag<CR>
```
